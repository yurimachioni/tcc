É necessário ter ambas chipmunk e SDL2 devidamente instaladas e linkadas para compilar o código.
a versão da chipmunk utilizada foi a 6.2.1
a versão da SDL2 utilizada foi a 2.0.1
a versão da SDLimage utilizada foi a 2.0.0
neste mesmo diretório há outros .txt com algumas dicas para compilar as bibliotecas.

Para execução do jogo basta ter as DLLs contidas nesta pasta no mesmo diretório do .exe
o jogo foi testado em Windows 7 em alguns hardwares diferentes, a apresenta taxa de quadros consistente

Os controles do jogo são:

->  move personagem para a direita (seta para a direita)
<-  move personagem para a esquerda (seta para a esquerda)
^   mira arma para cima (seta para cima)
3   dispara arma
2   pula
R   reseta partida
0   ganha vida

O jogo não têm condição de vitória nem limite de gerações.
Caso o jogador perca toda a vida, a partida é resetada (inclusive inimigos).
